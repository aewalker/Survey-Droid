  • 4/6
    • Met with Naomi.
    • Designed survey model:
      • 5 level system - highest is survey level
      • survey has questions
      • question has choices (as in multiple choice; a question without choices is interpreted as a free-response question)
      • question has branches which create control flow; a branch is a link from a parent question to the next question
      • branches have conditions, which check previous questions; if all conditions for a branch (can be 0) are true, then that branch is followed
      • once there are no more possible branches, the survey ends
